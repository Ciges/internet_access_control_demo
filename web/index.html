Hello,

In this directory you will find 6 scripts which make part of the load tests collection for a comparative performance study I have made between MongoDB 2.2.0rc0 and MySQL 5.0.28 (with MyISAM tables).

The scripts included are:

  • test4_mongo.php :  MongoDB read test. Search and show data for a random user
  • test4_mysql.php :  MySQL read test. Search and show data for a random user
  • test5_mongo.php :  MongoDB write test. Write a random user
  • test5_mysql.php :  MySQL write test. Write a random user
  • test6_mongo.php :  MongoDB read/write test. This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show data for a random user (read test)
    • Write a new random user in the database (write test)
  • test6_mysql.php :  MySQL read/write test. This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show data for a random user (read test)
    • Write a new random user in the database (write test)
  • test7_mongo.php :  MongoDB read test on NonFTP Logs records collection (millions of documents). Search and show logs records starting for a random date (limited to 10)
  • test7_mysql.php :  MySQL read test on NonFTP Logs records table (millions of records). Search and show logs records starting for a random date (limited to 10)
  • test8_mongo.php :  MongoDB write test on NonFTP Logs records collection (millions of documents). Write a (predefined) log record in Non FTP Logs collection
  • test8_mysql.php :  MySQL write test on NonFTP Logs records collection (millions of documents). Write a (predefined) log record in Non FTP Logs collection
  • test9_mongo.php :  MongoDB read/write test on NonFTP Logs records collection (millions of documents). This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show logs records starting for a random date (limited to 10) (read test)
    • Write a (predefined) log record in Non FTP Logs collection
  • test9_mysql.php :  MongoDB read/write test on NonFTP Logs records collection (millions of documents). This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show logs records starting for a random date (limited to 10) (read test)
    • Write a (predefined) log record in Non FTP Logs collection
  • test10_mysql.php :  MySQL read test on NonFTP Logs records table (millions of records). Search and show logs records for a random domain
  • test11_mysql.php :  MySQL read/write on NonFTP Logs records table (millions of records). This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show logs records for a random domain
    • Write a (predefined) log record in Non FTP Logs collection
  • test8_mysql.php :  MySQL write test on NonFTP Logs records table (millions of records). Write a (predefined) log record in Non FTP Logs collection
  • test10_mongo.php :  MongoDB read test on NonFTP Logs records collection (millions of documents). Search and show logs records for a random domain
  • test11_mongo.php :  MongoDB read/write test on NonFTP Logs records collection (millions of documents). This script makes one of two read or write actions. The read test is made 80% of times, the write one the 20%.
    • Search and show logs records for a random domain
    • Write a (predefined) log record in Non FTP Logs collection
  • test8_mongo.php :  MongoDB write test on NonFTP Logs records collection (millions of documents). Write a (predefined) log record in Non FTP Logs collection
