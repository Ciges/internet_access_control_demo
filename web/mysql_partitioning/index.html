In this directory you will find scripts which make part of the load tests collection for a comparative read & write performance tests I have made over a MySQL table (MyISAM) with millions of log records and the same MySQL table but using partitioning (partitioned by month):

The scripts included are:

  • write_a_2012_log_record.php :  Writes a predefined Non FTPlog record (only 'datetime' record is randomly generated) in "NonFTP_Access_log" table.
  • write_a_2012S52_log_record.php :  Writes a predefined Non FTPlog record (only 'datetime' record is randomly generated but only for last week of 2012) in "NonFTP_Access_log" table.
